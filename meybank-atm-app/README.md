# Meybank ATM Secure Menu Web Application

A modern web application simulating an ATM menu for Meybank, built for a hackathon demo.

## Features
- Balance Enquiry
- Cash Deposit
- Withdrawal
- Third Party Transfer
- Transactions
- Change ATM Card PIN
- Logout

## Tech Stack
- React (frontend)
- Node.js + Express (backend, in-memory demo)

## How to Run
1. Install dependencies in both `client` and `server` folders.
2. Start the backend: `npm start` in `server`.
3. Start the frontend: `npm start` in `client`.

---

This project is for demonstration purposes only. No real banking operations are performed.
